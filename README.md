# Bike-Accesaries-Sales-Analysis-Interactive-Dashboard-SQL-Power-BI-

##  Project Overview
This project showcases an **end-to-end sales data analysis workflow** using **SQL for data analysis** and **Power BI for visualization**.  
The objective is to extract meaningful business insights from raw sales data and present them through an **interactive, decision-ready dashboard**.

This project is designed to demonstrate **real-world data analyst skills**, including querying, KPI creation, trend analysis, and dashboard storytelling.

The Power BI dashboard provides:
-  kPI cards for **Sales, Revenue, Profit**
-  Line charts for **monthly & yearly trends**
-  Bar charts for **product & category performance**
-  Regional sales distribution analysis
-  Interactive slicers (Date, Product, Category, Region)
-  Drill-down analysis for deeper insights
